# CheckStock

## サイト概要
### サイトテーマ
出先から自分の**ストック状況**を確認でき、必要に応じて他者にも共有できるアプリケーション

### テーマを選んだ理由
料理を作ろうとした際に現在どの調味料や食材が残っているか、何を買い足せばいいかを出先からでも把握できたら便利だと考え、このテーマにしました。

### ターゲットユーザ
- ストック状況を把握したい人
- ストックを可視化したい人

### 主な利用シーン
- 冷蔵庫の中身を確認せず買い物に来たとき
- 買い物メモに不備がないか確認するとき
- 使い方次第でスキンケア用品や日用品などにも応用可能です。
